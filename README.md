# DashBoard
Инфромационная панель для Raspberry pi


## Доступные плагины
1. Часы (block_time)
2. Календарь (block_calendar)
3. Погода от OpenWeaterMap (block_openweatermap)
4. Новости от Яндекса (block_yandexnew)
5. Голос YandexSpeech - озвучивает полученный текст (block_voice)
6. Аггрегатор текста - формирует текст для озвучки (block_textaggregator)
